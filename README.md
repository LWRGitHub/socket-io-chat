# Socket.io Chat
This is a web app that uses Socket.io in order to send messages back & forth without refreshing the page. I hope you enjoy using it!


## Tech used
- JaveScript
- Node.js
- HTML/CSS
- Socket.io
- express


## Install/Setup

### Install
***make sure you have***
```
npm install
```

### Run
***@ localhost:3000***
```
node index.js
```
